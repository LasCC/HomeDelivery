# Partie technique

HomeDelivery utilises les technologies suivantes :

### Back-end :

- Node-JS, ExpressJS

- Python (POO)

- MongoDB

- CentOS (Serveur AWS)

- SQLite

### Front-end :

- ReactJS

- Framework CSS (Material-ui)

- Figma/Adobe XD pour le prototype d'application

- Mapbox JS

- (Mobile) React Native

- Chart-JS
